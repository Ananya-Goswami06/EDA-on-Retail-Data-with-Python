# EDA-on-Retail-Data-with-Python

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Online Retail dataset using Python. The objective is to understand sales patterns, customer purchasing behavior, top-selling products, country-wise performance, and identify potential outliers in the data.

## Tools and Libraries

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Dataset

The dataset contains transactional records from an online retail store, including:

* Invoice Number
* StockCode
* Product Description
* Quantity
* Invoice Date
* Unit Price
* Customer ID
* Country

## Tasks Performed

1. Data loading and inspection
2. Data cleaning and preprocessing
3. Statistical summary of the dataset
4. Data visualization
5. Monthly and weekly sales trend analysis
6. Top-selling products analysis
7. Country-wise sales analysis
8. Outlier detection
9. Key business insights and conclusions

## Key Findings

1. Identified seasonal fluctuations and peak sales periods.
2. Determined the most frequently purchased products.
3. Evaluated country-wise sales contributions and market concentration.
4. Detected and examined outliers that could affect business insights.
5. Uncovered purchasing patterns through time-based sales analysis.
